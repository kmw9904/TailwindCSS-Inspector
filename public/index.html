주요 기능

자동 CSS 분석 및 변환

페이지의 요소 위에 마우스를 올리면 하이라이트되고, 클릭 시 해당 CSS가 Tailwind 유틸리티 클래스로 변환됩니다.

AI 기반 상세 설명

OpenAI GPT-4o를 활용하여 CSS 변환 과정을 단계별로 상세하게 설명해 드립니다.

실시간 CSS → Tailwind 변환

일반 CSS를 입력하면 AI가 이를 Tailwind CSS 클래스로 즉시 변환해줍니다.

반응형 & 드래그 UI

사이드바를 드래그하여 크기를 조절하고, 토글 버튼으로 쉽게 열고 닫을 수 있습니다.
